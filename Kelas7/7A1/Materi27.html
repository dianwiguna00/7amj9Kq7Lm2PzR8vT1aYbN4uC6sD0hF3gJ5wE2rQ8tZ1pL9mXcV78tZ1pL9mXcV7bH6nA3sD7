Tabel Menyajikan data secara sistematis dalam baris dan kolom. Cocok untuk membandingkan harga, diskon, pajak, dan keuntungan. Grafik Batang Menampilkan data kuantitatif secara visual. Cocok untuk menunjukkan perbandingan harga sebelum dan sesudah diskon atau pajak. Diagram Garis Menunjukkan perubahan data dari waktu ke waktu, misalnya pertumbuhan tabungan dengan bunga.
✅ Contoh Soal Toko “SmartMart” menjual 3 barang berikut: Sepatu: harga awal Rp400.000, diskon 10%, pajak 11% Tas: harga awal Rp300.000, diskon 15%, pajak 11% Jaket: harga awal Rp500.000, diskon 20%, pajak 11% Tugas: Hitung harga setelah diskon untuk masing-masing barang. Hitung harga akhir setelah dikenai pajak. Sajikan hasilnya dalam bentuk tabel baru dan grafik batang.
  1. Latihan 1: Tabel Transaksi Buat tabel yang memuat data berikut: Nama barang Harga beli Harga jual Untung/Rugi Persentase untung/rugi Penyelesaian: Barang A Harga beli = 200.000 Harga jual = 240.000 Untung = 240.000 − 200.000 = 40.000 Persentase untung = 40.000 ÷ 200.000 = 0,2 = 20% Barang B Harga beli = 150.000 Harga jual = 135.000 Rugi = 135.000 − 150.000 = −15.000 Persentase rugi = 15.000 ÷ 150.000 = 0,1 = 10% Barang C Harga beli = 300.000 Harga jual = 330.000 Untung = 330.000 − 300.000 = 30.000 Persentase untung = 30.000 ÷ 300.000 = 0,1 = 10%
  2. Latihan 2: Grafik Tabungan Seorang siswa menabung Rp1.000.000 dengan bunga tunggal 5% per tahun. Buat grafik garis yang menunjukkan pertumbuhan tabungan selama 5 tahun. Penyelesaian: Bunga per tahun = 5% × 1.000.000 = 0,05 × 1.000.000 = 50.000 Tahun ke-1 = 1.000.000 + 50.000 = 1.050.000 Tahun ke-2 = 1.000.000 + 2 × 50.000 = 1.100.000 Tahun ke-3 = 1.000.000 + 3 × 50.000 = 1.150.000 Tahun ke-4 = 1.000.000 + 4 × 50.000 = 1.200.000 Tahun ke-5 = 1.000.000 + 5 × 50.000 = 1.250.000 Data grafik garis = tahun pada sumbu horizontal = nilai tabungan pada sumbu vertikal
  3. Latihan 3 (Kontekstual) Sebuah toko memberikan diskon dan mengenakan pajak pada 4 jenis barang. Buat tabel dan grafik batang yang menunjukkan: Harga awal Harga setelah diskon Harga akhir setelah pajak Penyelesaian: Barang 1 Harga awal = 500.000 Diskon = 10% × 500.000 = 50.000 Harga setelah diskon = 450.000 Pajak = 10% × 450.000 = 45.000 Harga akhir = 495.000 Barang 2 Harga awal = 400.000 Diskon = 15% × 400.000 = 60.000 Harga setelah diskon = 340.000 Pajak = 10% × 340.000 = 34.000 Harga akhir = 374.000 Barang 3 Harga awal = 300.000 Diskon = 20% × 300.000 = 60.000 Harga setelah diskon = 240.000 Pajak = 10% × 240.000 = 24.000 Harga akhir = 264.000 Barang 4 Harga awal = 200.000 Diskon = 25% × 200.000 = 50.000 Harga setelah diskon = 150.000 Pajak = 10% × 150.000 = 15.000 Harga akhir = 165.000 Data grafik batang = jenis barang pada sumbu horizontal = harga akhir pada sumbu vertikal
